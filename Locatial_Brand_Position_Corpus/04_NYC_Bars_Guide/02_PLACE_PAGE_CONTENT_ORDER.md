# Place Page Content Order

The user should be able to make an initial decision in approximately five seconds.

## Layer 1: Signal

1. Hero photograph
2. Place name
3. Defining headline
4. Neighborhood and venue type
5. One-sentence verdict
6. Primary action: save, route or add to night

## Layer 2: Decision strip

- Best for
- Go at
- Expected spend
- Sound
- Group size
- Access
- Typical stay

Use human language for sound:

- Quiet
- Conversational
- Animated
- Loud
- Music-led

## Layer 3: Editorial read

A concise description covering threshold, room, people, time and point of view.

## Layer 4: What to order

- Defining drink
- Safer alternative
- Food item
- Non-alcoholic option
- Menu last checked

## Layer 5: Know before you go

- reservation reality;
- queue;
- group constraints;
- dress expectations where relevant;
- accessibility;
- outdoor space;
- payment policy;
- last verified date.

## Layer 6: Place in context

- map;
- neighborhood;
- what to do before;
- where to go after;
- walking relationship;
- nearby contrast;
- subway and late-night transport.

## Layer 7: Deeper layer

- origin;
- ownership;
- building history;
- cultural significance;
- archival imagery;
- related stories;
- creator notes.

## Principle

Help the user decide first. Reward curiosity second.
