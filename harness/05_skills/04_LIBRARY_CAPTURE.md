# Skill — Library & Capture

Use this for Library, captures, notes, media, links, and reusable material.

## Model

Global Library → Stop Context → Stop Editor.

## Must support

- Global Library items.
- Stop-specific notes.
- Used / unused states.
- Drag or click to move material into context.
- Drag or click from context into editor.
- Search/filter later.

## Do not

- Treat all notes as publishable content.
- Force users to classify every capture immediately.
- Lose the difference between global material and page-level working notes.
