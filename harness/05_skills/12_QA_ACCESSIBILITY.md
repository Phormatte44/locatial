# Skill — QA / Accessibility

Use this for testing, keyboard behavior, mobile, and usability checks.

## Check

- Keyboard navigation.
- Click targets.
- Empty states.
- Error states.
- Mobile preview.
- Basic color contrast.
- Long titles and overflowing content.
- Missing location.
- Missing image.
- Unassigned Stop.
- Used / unused note states.

## Do not

- Only test the happy path.
