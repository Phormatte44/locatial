# Skill — Place Node Model

Use this for location, place search, duplicate detection, and reusable geography.

## Rule

A Place Node is created once and referenced many times.

## Place Node holds

- Name
- Type
- Coordinates / geometry
- Address if relevant
- Aliases
- Sources
- Confidence
- Last checked

## Stop holds

- Why this place matters in this project.
- Story-specific copy.
- Local media choice.
- Optional camera framing.
