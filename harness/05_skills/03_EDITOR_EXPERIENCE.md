# Skill — Editor Experience

Use this when building Story Stops, Stop Editor, Research Tray, Spatial Context, or Preview.

## Optimize for

- Rough story quickly.
- Add material from Library.
- Write one stop at a time.
- Keep map available but not dominant.
- Assign location and camera framing only when useful.
- Preview the actual audience journey.

## Do not

- Turn the editor into Notion.
- Turn the editor into ArcGIS.
- Overload the right rail with settings.
- Make the map permanently dominant by default.
