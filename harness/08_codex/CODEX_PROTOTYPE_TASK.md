## Visual instruction

Use neutral dark mode.

Do not draw from previous Locatial mockups, globe explorations, neon styling, or brand experiments.

This first build is for interaction validation only.

## Build scope

Create the first working prototype only.

Must include:

- One Creator Studio route.
- Three-column editor.
- Dummy Story Stops.
- Dummy Stop Editor.
- Dummy Research Tray.
- Dummy Global Library.
- Spatial Context drawer.
- Camera fields: zoom, pitch, bearing.
- Preview mode.
- No backend.
- No auth.
- No Supabase.
- No production dataset import.

## Output requirements

After building, report:

- Files changed.
- How to run locally.
- What interactions work.
- What is intentionally fake.
- What should be validated next.
