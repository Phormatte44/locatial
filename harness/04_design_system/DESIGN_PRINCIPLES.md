# Design Principles

- Editorial before technical
- Calm, precise, premium
- Content-led
- Map as lens, not center
- Strong typography
- Minimal controls
- Progressive disclosure
- One controlled accent
- Avoid dashboard clutter

## Prototype visual constraint

First prototype uses neutral dark mode with shadcn/ui and Obra shadcn_ui kit Pro edition 1.13.2.

Do not judge final visual design from the first prototype.
