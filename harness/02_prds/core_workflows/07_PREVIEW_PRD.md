# PRD — Preview

## Purpose

Preview is a live rehearsal of the audience experience.

## Device modes

- Phone
- Tablet
- Desktop

## Required behavior

- Show cover
- Move through stops
- Map updates with stop progression
- Apply Stop camera framing where available
- Open place detail
- Show route if enabled
- Show spatial layers if enabled
- Jump from preview back to editable stop

## Principle

Preview is not the final step. It is part of authoring.
