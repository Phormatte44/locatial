# PRD — Library

## Purpose

Library is what the creator has.

It stores reusable and uploaded material.

## Content types

- Places
- Captures
- Notes
- Photos
- Videos
- Links
- Datasets
- Project references

## Required functions

- Search across all library material
- Filter by content type
- Import datasets
- Retrieve captures inside projects
- Move library material into project context
- Resolve captures into Place Nodes

## Principle

The Library is a spatial archive, not an admin backlog.
