# PRD — Place Node

## Purpose

A Place Node is one reusable spatial object.

Examples:

- Building
- Bench
- Graffiti wall
- Café
- Park
- Neighborhood
- Borough
- City
- F1 circuit

## Shared truth

- Name
- Type
- Geometry
- Address if relevant
- Coordinates
- Aliases
- Sources
- Confidence
- Last checked
- Created by
- Reuse count

## Project-specific meaning

Each project can attach its own snippet, verdict, image, or interpretation to the same Place Node.

## Duplicate handling

- Suggest existing matches
- Allow create anyway
- Mark possible duplicate
- Merge later
