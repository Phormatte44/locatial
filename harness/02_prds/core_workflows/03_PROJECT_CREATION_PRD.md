# PRD — Project Creation

## Purpose

Projects are what creators make.

Project types:

- Story
- Guide
- Collection

## Starting material

A creator can start from:

- Empty project
- Places
- Captures
- Dataset
- Existing notes
- Photos/media

## Required flow

1. Choose project type
2. Add title
3. Define audience promise
4. Choose Channel / publishing identity when available
5. Choose optional starting material
6. Open editor

## Principle

Do not force classification before creation.
