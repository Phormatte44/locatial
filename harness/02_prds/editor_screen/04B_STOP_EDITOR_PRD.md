# PRD — Stop Editor

## Purpose

The Stop Editor is where the creator writes and assembles content for one stop.

## Default blocks for every new stop

1. Image block
2. Headline block
3. Text block

## Supported blocks

- Text
- Heading
- Quote
- Image
- Gallery
- Video placeholder
- Place block
- Chart block
- Dataset reference block
- Spatial reference block

## Text formatting

Support:

- Bold
- Italic
- Link
- Bullets
- Numbered lists
- Quote

Avoid early:

- Custom fonts
- Text colors
- Layout columns
- Freeform page design
