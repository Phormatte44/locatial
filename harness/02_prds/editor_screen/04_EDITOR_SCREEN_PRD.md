# PRD — Editor Screen

## Purpose

The editor is the primary desktop workspace for shaping, writing, and checking a project.

## Layout

Three persistent regions:

1. Story Stops / Structure rail
2. Stop Editor / main canvas
3. Research Tray / contextual material

## Supporting surfaces

- Spatial Context
- Preview
- Publish

## Principle

The creator roughs the story, gathers evidence, writes each stop, and previews the audience journey.
