# PRD — Research Tray

## Purpose

The Research Tray supplies evidence and working material.

## Two zones

### Stop Context

Notes and material relevant to the selected stop.

### Global Library

Searchable access to previous captures, notes, media, links, places, and datasets.

## Core flow

Library item → Stop Context → Stop Editor

## Required functionality

- Search library
- Filter material
- Move item to stop context
- Drag context note into editor
- Mark item used
- Show unused material
- Add new stop-level note
