# PRD — Story Stops

## Purpose

Story Stops shape the arc of the project.

A Stop is a project-level unit. It may reference a Place Node, but it does not have to.

## Stop states

- Empty
- Outlined
- Researching
- Drafting
- Review
- Done

## Place assignment states

- Unassigned
- Candidate
- Assigned

## Required functionality

- Add stop
- Rename stop
- Reorder stop
- Group stops
- Collapse groups
- Drag dataset feature into stop
- Drag library material into stop
- Show status
- Show missing location warning
- Open Spatial Context

## Do not

- Treat every Stop as a Place
- Require location before writing
- Expose internal ontology too early
